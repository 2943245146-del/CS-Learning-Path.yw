拷贝构造函数:
    person(const person &a)
    深拷贝（重新在堆区创建一个内存）可以解决浅拷贝（编译器默认实现）带来的问题（堆开辟）
初始化列表
        初始化属性
    eg. person():m_A(10),m_B(20),m_C(30)
        {
        }
        or
        person(int a,int b,int c):m_A(a),m_B(b),m_C(c)
        {
        }
静态成员变量
    所有对象共享一份数据
    编译阶段就分配内存
    类内声明类外初始化
        类外eg. int person::m_A=100;
静态成员函数
    所有对象共享一个函数
    只能调用静态成员变量
    可以通过类名访问
        person::func();
成员变量和成员函数是分开存储的，只有非静态成员变量属于类的对象上（增加sizeof（）值）
